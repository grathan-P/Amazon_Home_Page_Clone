# Amazon Homepage UI Clone

A static Amazon homepage UI clone built using **HTML and CSS**.  
This project was created to strengthen my understanding of layout, spacing, and visual hierarchy.

## Project Overview
This is a frontend-only clone of the Amazon homepage.  
The focus of this project was on:
- Structuring layouts using HTML
- Styling with CSS
- Flexbox-based alignment
- Visual consistency and spacing

> Note: This project is currently **static and non-responsive** and does not include JavaScript functionality.

## Technologies Used
- HTML5  
- CSS 

## Features
- Amazon-style navigation bar  
- Banner section layout  
- Product grid sections  
- Clean UI structure
